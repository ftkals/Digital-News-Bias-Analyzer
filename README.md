# Digital News Bias Analyzer

A college mini project developed as part of the **Statistics for AI & Machine Learning (STAIML)** coursework. This project explores media bias, engagement patterns, and news reliability using statistical analysis, data visualization, and basic machine learning techniques in both Python and R.

## Project Overview

With the increasing amount of information available online, understanding how news is presented and consumed has become important. This project analyzes a dataset of news headlines collected from multiple media sources and investigates how factors such as headline tone, source reliability, engagement metrics, and content categories influence audience reach and impact.

The analysis was performed using Python and R to compare results across both environments and gain hands-on experience with data analytics workflows.

## Objectives

* Analyze news headlines from different media sources
* Study engagement and reach patterns across categories
* Detect positive, negative, and neutral headline tones
* Explore relationships between headline characteristics and impact
* Compare trusted and verify-classified news sources
* Apply basic machine learning models for impact prediction

* Exploratory Data Analysis (EDA)
* Outlier Detection
* Headline Tone Classification
* Correlation Analysis
* Engagement Volatility Analysis
* Reliability Distribution Analysis
* Data Visualization
* Impact Score Prediction using:

  * Linear Regression
  * Random Forest Regressor

## Major Findings

Some interesting observations from the project:

* Global Politics headlines showed the highest engagement levels.
* Negative-toned headlines generally achieved greater reach than positive headlines.
* Headline length had only a weak relationship with overall impact.
* Trusted and verify-classified sources displayed different engagement patterns.
* Linear Regression performed better than Random Forest for this dataset.
